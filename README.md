# NextContact
A simple Contact Manager landing page built using Next.js (App Router) and Tailwind CSS.  
This project is created to understand the fundamentals of Next.js routing, components, and basic UI structuring.

# Features
- Modern UI built with Tailwind CSS
- Responsive layout
- Optimized images using `next/image`
- Built using Next.js App Router
- Beginner-friendly project structure

# Tech Stack
- Next.js 15
- React
- TypeScript
- Tailwind CSS
